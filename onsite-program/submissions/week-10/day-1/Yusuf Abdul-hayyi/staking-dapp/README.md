# Staking DApp

React + Vite staking UI using wagmi, viem and RainbowKit.

## Environment

Create a `.env` file with:

```
VITE_WALLET_CONNECT_PROJECT_ID=your_walletconnect_id
VITE_STAKING_CONTRACT=0xYourStakingContractAddress
```

## Scripts

- dev: `yarn dev`
- build: `yarn build`
- preview: `yarn preview`
